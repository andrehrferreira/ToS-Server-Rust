# Dual Blade Skillset

## Overview

Dual Blade focuses on dual-wielding and rapid strikes. This skillset provides abilities for fast, consecutive attacks, dual-wield bonuses, and high attack speed. Perfect for players who prefer fast-paced, high-damage melee combat.

## Skillset Theme

**Combat Style:** Fast Melee DPS
**Primary Role:** Damage Dealer
**Secondary Role:** Burst Damage
**Resource:** Stamina
