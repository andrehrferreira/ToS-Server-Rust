# Bardic Arts Skillset

## Overview

Bardic Arts utilizes music and song to buff allies, debuff enemies, and provide support through various melodies. This skillset provides group support, crowd control through music, and healing capabilities. Perfect for players who prefer support roles with musical themes.

## Skillset Theme

**Combat Style:** Support/Bard
**Primary Role:** Buffer / Support
**Secondary Role:** Crowd Control
**Resource:** Mana
