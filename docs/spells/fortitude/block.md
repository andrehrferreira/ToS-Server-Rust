# Block

## Overview

Block raises the shield to block incoming attacks, providing massive damage mitigation for a brief moment. This defensive ability requires timing but provides exceptional protection.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Target Self
**Target Type:** Self
**Range:** Self
**Cast Time:** Instant
**Cooldown:** 6 seconds
**Resource Cost:** 15 Stamina
**Block Window:** 1.0 seconds

## Effects

**Block:**
- Damage Reduction: +80% (during window)
- Blocks all damage types
- Requires shield equipped
- Visual block effect

**Additional Effects:**
- High skill requirement
