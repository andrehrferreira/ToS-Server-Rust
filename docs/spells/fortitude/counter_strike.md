# Counter Strike

## Overview

Counter Strike allows the warrior to counter-attack immediately after blocking an attack, dealing damage back to the attacker. This defensive attack ability provides both protection and offense.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Single Target (Reactive)
**Target Type:** Enemy
**Range:** Melee (2 units)
**Cast Time:** Instant (reactive)
**Cooldown:** 10 seconds
**Resource Cost:** 20 Stamina

## Effects

**Counter:**
- Physical Damage: 180% of Attack Power
- Triggers automatically after successful block
- Cannot be interrupted
- Visual counter effect

**Block Requirement:**
- Must successfully block an attack
- Triggers within 0.5 seconds of block
- High skill requirement
- Defensive offense ability

**Additional Effects:**
- Visual counter strike
- High damage potential
- Essential defensive ability
- Core tank skill
