# Unbreakable

## Overview

Unbreakable grants complete immunity to all status effects and crowd control abilities, allowing the warrior to continue fighting unimpeded. This rare defensive ability provides absolute control immunity.

## Spell Details

**Rarity:** Rare (Blue)
**Spell Type:** Target Self
**Target Type:** Self
**Range:** Self
**Cast Time:** Instant
**Cooldown:** 90 seconds
**Resource Cost:** 60 Mana
**Duration:** 15 seconds

## Effects

**Immunity:**
- Status Effect Immunity: 100%
- Crowd Control Immunity: 100%
- Cannot be stunned, slowed, or controlled
- Visual unbreakable effect

**Additional Effects:**
