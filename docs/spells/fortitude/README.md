# Fortitude Skillset

## Overview

Fortitude focuses on defensive techniques and damage mitigation. This skillset provides abilities to increase resistance, absorb damage, generate threat, and protect allies. Perfect for tank roles and players who prefer defensive playstyles.

## Skillset Theme

**Combat Style:** Tank/Defense
**Primary Role:** Tank
**Secondary Role:** Protector
**Resource:** Stamina/Mana hybrid

## Synergies

**Good Combinations:**
- **Warcraft**: Tank and melee DPS combination
- **Lifebinding**: Tank and healer combination
- **Auraweaving**: Additional buffs and utility

**Playstyle:**
- Absorb damage for party
- Generate threat and control enemies
- Protect allies
- High survivability focus
