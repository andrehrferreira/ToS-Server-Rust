# Auraweaving Skillset

## Overview

Auraweaving focuses on buffs, debuffs, and mobility enhancements. This skillset provides abilities to strengthen allies, weaken enemies, and enhance movement capabilities. Perfect for support roles and players who want versatile utility abilities.

## Skillset Theme

**Combat Style:** Support/Utility
**Primary Role:** Buffer / Debuffer
**Secondary Role:** Mobility Support
**Resource:** Mana

## Synergies

**Good Combinations:**

- **Elementalism**: Mana support for mages
