# Dimensional Rift

## Overview

Dimensional Rift creates a portal that allows instant travel between two locations. This rare ability provides advanced mobility and positioning options.

## Spell Details

**Rarity:** Rare (Blue)
**Spell Type:** Target Area (Two Locations)
**Target Type:** Ground Location
**Range:** 30 units (each portal)
**Cast Time:** 3.0 seconds
**Cooldown:** 90 seconds
**Resource Cost:** 100 Mana
**Portal Duration:** 60 seconds

## Effects

**Portal Creation:**
- Creates two connected portals
- Instant travel between portals
- Can be used by allies
- Visual portal effect

**Additional Effects:**
- Advanced mobility option
- Group positioning advantage
- High utility ability
- Strategic positioning tool
