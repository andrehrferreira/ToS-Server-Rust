# Dispel

## Overview

Dispel removes beneficial buffs from enemies, reducing their combat effectiveness. This utility ability provides crowd control by removing enemy advantages.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Single Target
**Target Type:** Enemy
**Range:** 20 units
**Cast Time:** Instant
**Cooldown:** 12 seconds
**Resource Cost:** 30 Mana

## Effects

**Dispel:**
- Removes all buffs from target
- Cannot remove certain boss buffs
- Instant effect
- Visual dispel effect

**Additional Effects:**
- Reduces enemy effectiveness
- Essential utility ability
- High support utility
- Crowd control tool

**Level 1:** Base (removes all buffs)
**Level 2:** Can remove 1 additional buff type
