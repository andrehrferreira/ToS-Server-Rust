# Mana Shield

## Overview

Mana Shield creates a protective barrier that absorbs damage using mana instead of health. This defensive ability provides survivability at the cost of mana resources.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Target Self
**Target Type:** Self
**Range:** Self
**Cast Time:** Instant
**Cooldown:** 15 seconds
**Resource Cost:** 50 Mana (activation)
**Duration:** 60 seconds (toggle)

## Effects

**Shield:**
- Damage Absorption: 200% of Magic Power
- Absorbs damage using mana (1:1 ratio)
- Shield breaks when mana depleted
- Visual shield effect

**Additional Effects:**
- Mana-based defense
- Can be toggled on/off
- High survivability
- Resource management required
