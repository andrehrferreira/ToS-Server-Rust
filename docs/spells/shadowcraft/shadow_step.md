# Shadow Step

## Overview

Shadow Step instantly teleports the caster behind the target, allowing for immediate backstab positioning. This mobility ability is essential for shadowcraft builds, providing positioning advantage.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Single Target
**Target Type:** Enemy
**Range:** 20 units
**Cast Time:** Instant
**Cooldown:** 12 seconds
**Resource Cost:** 30 Stamina

## Effects

**Teleportation:**
- Instant teleport behind target
- Maintains stealth if active
- Cannot be interrupted
- Brief invulnerability (0.2 seconds)

**Additional Effects:**
- Visual shadow step effect
- Positioning advantage
- Can be used while stealthed
- Essential mobility ability
