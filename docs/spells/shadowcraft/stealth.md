# Stealth

## Overview

Stealth renders the player invisible, allowing them to move undetected and set up devastating backstab attacks. This fundamental stealth ability is essential for shadowcraft gameplay, providing positioning advantages and enabling critical strike opportunities.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Target Self
**Target Type:** Self
**Range:** Self
**Cast Time:** 1.0 seconds
**Cooldown:** 30 seconds
**Resource Cost:** 40 Stamina
**Duration:** 60 seconds (or until broken)

## Effects

**Stealth:**
- Invisibility: Complete visual invisibility
- Detection Range: Reduced by 80%
- Movement Speed: -20% while in stealth
- Break Conditions: Attacking, taking damage, using certain abilities

**Benefits:**
- Cannot be targeted by enemies (unless detected)
- Backstab attacks deal 3x damage
- Critical strike chance +50% from behind
- Can move through enemy lines

**Limitations:**
- Slower movement speed
- Breaks on any attack
- Breaks on taking damage
- Some abilities break stealth

## Visual Effects

**Activation:**
- Fade out animation
- Shadow particles
- Invisibility shimmer

**Stealth State:**
- Invisible to enemies
- Slight shimmer effect (optional)
- Shadow trail (optional)

**Breaking:**
- Fade in animation
- Shadow burst
- Reveal effect

**Sound Effects:**
- Stealth activation sound
- Ambient shadow sounds
- Break sound

## Usage Notes

**Best Used For:**
- Positioning for backstab attacks
- Escaping dangerous situations
- Scouting enemy positions
- Setting up ambushes

**Combat Tips:**
- Use before engaging enemies
- Position behind targets for backstab
- Combine with Shadow Step for positioning
- Use Vanish for emergency escape

**Stealth Synergy:**
- Backstab deals massive damage from stealth
- Shadow Step teleports while maintaining stealth
- Assassinate requires stealth for maximum damage

## Upgrades

**Level 1:** Base duration (60 seconds), -20% movement speed
**Level 2:** +10s duration, -15% movement speed penalty
**Level 3:** +20s duration, -10% movement speed penalty
**Level 4:** +30s duration, -5% movement speed penalty
**Level 5:** +40s duration, no movement speed penalty, +10% detection reduction
