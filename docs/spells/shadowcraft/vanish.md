# Vanish

## Overview

Vanish instantly enters stealth and removes all debuffs, providing an emergency escape option. This rare ability provides instant stealth and debuff removal, essential for survival.

## Spell Details

**Rarity:** Rare (Blue)
**Spell Type:** Target Self
**Target Type:** Self
**Range:** Self
**Cast Time:** Instant
**Cooldown:** 60 seconds
**Resource Cost:** 40 Stamina
