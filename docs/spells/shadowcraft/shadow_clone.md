# Shadow Clone

## Overview

Shadow Clone creates a decoy that distracts enemies and can be used to teleport to. This rare ability provides both crowd control and advanced positioning options.

## Spell Details

**Rarity:** Rare (Blue)
**Spell Type:** Target Area
**Target Type:** Ground Location
**Range:** 15 units
**Cast Time:** 1.5 seconds
**Cooldown:** 30 seconds
**Resource Cost:** 50 Mana
**Clone Duration:** 10 seconds

## Effects

**Clone:**
- Creates decoy at target location
- Enemies attack clone
- Clone has 50% of caster's health
- Visual clone effect

**Teleportation:**
- Can teleport to clone location
- Instant teleportation
- Maintains stealth if active
- Can be used while clone active

**Additional Effects:**
- Visual clone creation
- Decoy distraction
- Positioning advantage
- Essential advanced ability
