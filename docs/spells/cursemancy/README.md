# Cursemancy Skillset

## Overview

Cursemancy utilizes dark magic and curses from the abyss to inflict damage and apply debilitating debuffs to enemies. This skillset provides powerful dark magic attacks, curse effects, and damage over time abilities. Perfect for players who prefer dark, offensive magic.

## Skillset Theme

**Combat Style:** Dark Magic DPS
**Primary Role:** Damage Dealer
**Secondary Role:** Debuffer
**Resource:** Mana
