# Dark Shield

## Overview

Dark Shield creates a protective barrier of dark energy that absorbs incoming damage. This defensive ability provides survivability for cursemancy builds.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Target Self
**Target Type:** Self
**Range:** Self
**Cast Time:** Instant
**Cooldown:** 25 seconds
**Resource Cost:** 50 Mana
**Duration:** 45 seconds

## Effects

**Shield:**
- Damage Absorption: 250% of Magic Power
- Absorbs all damage types
- Shield breaks when absorption depleted
- Visual dark shield effect

**Additional Effects:**
- Visual shield effect
- High survivability increase
- Essential defensive ability
- Core defensive ability
