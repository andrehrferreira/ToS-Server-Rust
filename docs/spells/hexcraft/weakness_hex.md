# Weakness Hex

## Overview

Weakness Hex applies a hex that significantly reduces the target's damage output and combat effectiveness. This debuff hex provides enemy weakening.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Single Target
**Target Type:** Enemy
**Range:** 20 units
**Cast Time:** 1.5 seconds
**Cooldown:** 10 seconds
**Resource Cost:** 28 Mana
**Duration:** 25 seconds

## Effects

**Hex:**
- Damage Dealt Reduction: -35%
- Attack Power Reduction: -30%
- Cannot be dispelled by target
- Visual weakness hex effect

**Additional Effects:**
- Reduces enemy effectiveness
- Essential debuff hex
- High utility ability
- Core debuff ability
