# Confusion Hex

## Overview

Confusion Hex applies a hex that disorients the target, causing them to attack randomly and reducing their accuracy. This crowd control hex provides enemy disruption.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Single Target
**Target Type:** Enemy
**Range:** 18 units
**Cast Time:** 1.5 seconds
**Cooldown:** 12 seconds
**Resource Cost:** 32 Mana
**Duration:** 15 seconds

## Effects

**Hex:**
- Accuracy Reduction: -40%
- Attack Randomly: 30% chance
- Cannot target reliably
- Visual confusion hex effect

**Additional Effects:**
- Crowd control hex
- Essential control ability
- High utility ability
- Core hex ability
