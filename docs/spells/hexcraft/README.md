# Hexcraft Skillset

## Overview

Hexcraft focuses on mental manipulation and crowd control through hexes and curses. This skillset provides abilities to control enemy actions, apply mental debuffs, and manipulate combat through hex magic. Perfect for players who prefer strategic, control-oriented gameplay.

## Skillset Theme

**Combat Style:** Control/Hex Magic
**Primary Role:** Crowd Control
**Secondary Role:** Debuffer
**Resource:** Mana
