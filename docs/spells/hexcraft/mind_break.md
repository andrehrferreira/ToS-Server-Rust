# Mind Break

## Overview

Mind Break applies an extended control hex that prevents the target from using abilities and reduces their combat effectiveness for an extended duration. This rare control ability provides exceptional single-target control.

## Spell Details

**Rarity:** Rare (Blue)
**Spell Type:** Single Target
**Target Type:** Enemy
**Range:** 20 units
**Cast Time:** 2.5 seconds (channeled)
**Cooldown:** 50 seconds
**Resource Cost:** 90 Mana
**Duration:** 25 seconds

## Effects

**Hex:**
- Ability Lock: 100% (all abilities locked)
- Damage Dealt Reduction: -40%
- Cannot use any abilities
- Visual mind break effect

**Channeling:**
- Must channel for full duration
- Can be interrupted by damage
- Movement speed reduced by 70% while channeling

**Additional Effects:**
- Extended control hex
- Essential control ability
- High utility ability
- Core control ability
