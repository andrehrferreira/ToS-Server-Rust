# Mind Lock

## Overview

Mind Lock applies a hex that prevents the target from using certain abilities, locking their skills and reducing combat effectiveness. This control hex provides ability disruption.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Single Target
**Target Type:** Enemy
**Range:** 18 units
**Cast Time:** 1.8 seconds
**Cooldown:** 15 seconds
**Resource Cost:** 35 Mana
**Duration:** 20 seconds

## Effects

**Hex:**
- Ability Lock: 50% chance to lock abilities
- Locked abilities cannot be used
- Reduces combat effectiveness
- Visual mind lock effect

**Additional Effects:**
