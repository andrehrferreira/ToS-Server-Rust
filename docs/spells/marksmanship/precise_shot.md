# Precise Shot

## Overview

Precise Shot is a fundamental ranged attack that delivers a single, accurate arrow to the target. This spell provides reliable single-target damage with good range and accuracy.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Single Target
**Target Type:** Enemy
**Range:** 25 units
**Cast Time:** 0.5 seconds
**Cooldown:** 3 seconds
**Resource Cost:** 15 Mana

## Effects

**Damage:**
- Physical Damage: 120% of Attack Power
- Damage Type: Physical
- Can Critical Hit: Yes

**Additional Effects:**
- None (pure damage spell)

## Visual Effects

**Projectile:**
- Arrow projectile with trail effect
- Travels in straight line to target
- Impact effect on hit

**Animation:**
- Draw bow animation
- Release arrow animation
- Follow-through animation

**Sound Effects:**
- Bow draw sound
- Arrow release sound
- Impact sound on hit

## Usage Notes

**Best Used For:**
- Single target damage
- Finishing off low-health enemies
- Consistent ranged damage output
- Opening attacks from range

**Combat Tips:**
- Use from maximum range for safety
- Combine with Mark Target for increased damage
- Use Rapid Fire for burst damage combo
- Maintain distance while using

## Upgrades

**Level 1:** Base damage (120% Attack Power)
**Level 2:** +10% damage (132% Attack Power)
**Level 3:** +20% damage (144% Attack Power)
**Level 4:** +30% damage (156% Attack Power)
**Level 5:** +40% damage (168% Attack Power)
