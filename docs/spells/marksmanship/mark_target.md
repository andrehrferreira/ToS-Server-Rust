# Mark Target

## Overview

Mark Target places a mark on an enemy, increasing damage taken from all sources and providing vision of the target. This debuff ability enhances the effectiveness of all attacks against the marked enemy.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Single Target
**Target Type:** Enemy
**Range:** 25 units
**Cast Time:** Instant
**Cooldown:** 12 seconds
**Resource Cost:** 20 Mana
**Duration:** 30 seconds

## Effects

**Debuff:**
- Damage Taken Increase: +15%
- Applies to all damage sources
- Cannot be dispelled by target
- Visual mark on target

**Additional Effects:**
- Target visible through walls (brief)
- Mark visible to all allies
- Increases threat generation
- Stacks with other debuffs

## Visual Effects

**Casting:**
- Marking gesture
- Mark projection
- Mark application

**Active State:**
- Mark icon above target
- Glowing effect on target
- Mark visible to all players

**Sound Effects:**
- Marking sound
- Mark application sound
- Continuous mark sound

## Usage Notes

**Best Used For:**
- Increasing damage on priority targets
- Marking bosses and elites
- Enhancing team damage output
- Target identification

**Combat Tips:**
- Use on high-priority targets
- Mark before using high-damage spells
- Coordinate with team for focus fire
- Use on bosses for maximum effect

**Debuff Synergy:**
- Combines with other debuffs
- Enhances all damage sources
- Works with team coordination
- Stacks with vulnerability effects

## Upgrades

**Level 3:** +4% debuff (+19%), +10s duration
**Level 4:** +6% debuff (+21%), +15s duration
