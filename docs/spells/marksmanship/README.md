# Marksmanship Skillset

## Overview

Marksmanship focuses on ranged combat using bows and crossbows. This skillset provides powerful ranged attacks, area-of-effect abilities, and mobility options for maintaining distance from enemies. Perfect for players who prefer ranged combat and kiting strategies.

## Skillset Theme

**Combat Style:** Ranged DPS
**Primary Role:** Damage Dealer
**Secondary Role:** Crowd Control
**Resource:** Mana/Stamina hybrid

## Synergies

**Good Combinations:**
- **Shadowcraft**: Stealth and positioning for optimal shots
- **Auraweaving**: Mobility and buffs for kiting
- **Elementalism**: Ranged magic complement

**Playstyle:**
- Maintain distance from enemies
- Use mobility to kite
- Focus on single targets or area damage
- Utilize traps and marks for control
