# Piercing Arrow

## Overview

Piercing Arrow fires a powerful arrow that can pass through multiple enemies, dealing damage to all targets in its path. This spell excels at hitting multiple enemies in a line, making it effective against grouped enemies.

## Spell Details

**Rarity:** Common (Green)
**Spell Type:** Projectile
**Target Type:** Enemy or Direction
**Range:** 25 units
**Cast Time:** 1.0 seconds
**Cooldown:** 6 seconds
**Resource Cost:** 25 Mana

## Effects

**Damage:**
- Physical Damage: 140% of Attack Power (first target)
- Damage Reduction: 20% per enemy pierced
- Maximum Targets: 5 enemies
- Damage Type: Physical
- Can Critical Hit: Yes (per target)

**Piercing:**
- Passes through enemies
- Hits all enemies in line
- Damage decreases with each pierce
- Visual line indicator

## Visual Effects

**Casting:**
- Powerful draw animation
- Arrow charging effect
- Release animation

**Projectile:**
- Large arrow with trail
- Piercing visual effect
- Line indicator

**Impact:**
- Impact on each enemy
- Piercing through effect
- Final impact on last target

**Sound Effects:**
- Powerful draw sound
- Arrow release sound
- Multiple impact sounds
- Piercing sound

## Usage Notes

**Best Used For:**
- Hitting multiple enemies in line
- Clearing groups of enemies
- Maximum damage on first target
- Line-based area damage

**Combat Tips:**
- Position to hit multiple enemies
- Aim for enemy formations
- Use on grouped enemies
- Maximize first target damage

**Positioning:**
- Line up enemies for maximum hits
- Use terrain for positioning
- Consider enemy movement
- Aim for optimal line

## Upgrades

**Level 1:** Base damage (140% first, -20% per pierce), max 5 targets
**Level 2:** +10% first damage, +1 max target
**Level 3:** +20% first damage, -15% pierce reduction
**Level 4:** +30% first damage, -10% pierce reduction
**Level 5:** +40% first damage, -5% pierce reduction, +1 max target
